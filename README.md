<h3>Jax's Market</h3>

Aynamic and responsive grocery app offering groceries and home goods for instore/online ordering and pickup.
<br>
<br>
<br>
<h4>--- Tech ---</h4>
Backend: Java/Spring, SQL/MySQL 
<br><br>
Frontend: HTML5, CSS5, Typescript, Angular, Bootstrap
<br>
<br>
