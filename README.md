Jax's Market:

Aynamic and responsive grocery ordering app offering groceries and home goods for online ordering and pickup.
<br>
<br>
<br>

Built with:

Java<br>
HTML5<br>
CSS5<br>
JavaScript<br>
Typescript<br>

MySql<br>
Bootstrap<br>
Angular<br>
Spring Boot<br>
